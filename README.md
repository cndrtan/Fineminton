# Fineminton
Fineminton is an app for Badminton Player to to learn shot techniques.

## Tech Stack
**Platform:** iOS ``15.0``<br>
**Framework:** SwiftUI, WatchKit<br>
**IDE:** XCode ``v.13.4``<br>
